# Contributing

If you want to contribute, great! We make use of some command line utilities,
though there may be was to integrate these tools in your development environment
in other ways. PRs are required to fix any errors given off by these tools
before committing anything, so we can keep our code clean and easy to manage.

## Frontend

Required package manager: [bun](https://bun.com/docs/installation)

- Install: `bun install`
- Build: `bun build`
  - Dev mode: `bun dev`
- Format all code: `bun format`
- Lint: `bun lint`
- Test: `bun test`

## Backend

Required package manager: [composer](https://getcomposer.org/download/)

- Install: `composer install`
- Format all code: `composer format`
- Lint: `composer lint`
- Test: `composer test`
